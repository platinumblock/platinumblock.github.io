I. Introduction

Converse is a communication platform that allows users to be part of communities with coworkers, friends, and peers.

Each community has its own text and voice channels, notifications, permissions, and direct messaging lines. It's fully customizable and can work for any type of group. Users have status indicators indicating if they are online or not, and there is an @ system allowing users to get the attention of others quickly. Messages can also have reactions and replies.

This was a personal project I designed. I used Figma to make a high-fidelity prototype for a concept of this application.
